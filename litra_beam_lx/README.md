# Litra Beam LX Bridge (Home Assistant add-on)

Control Logitech Litra lights (Beam LX + back RGB, Beam, Glow) over Bluetooth
from Home Assistant. Multiple lights supported. MQTT broker credentials are
provided by the Supervisor automatically.

See [DOCS.md](DOCS.md) for full setup and configuration.

Part of [litra-ble-mqtt](https://github.com/alexvales/litra-ble-mqtt).
